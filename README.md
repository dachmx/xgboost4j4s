# xgboost4j4s

XGBoost4J with cross-version suffixes

## Development

### Prerequisites

* macOS
* Docker for Mac
* `g++-7` (installed by Homebrew, i.e. `brew install gcc`)

### How to Build and Test

```
$ make test
```

### How to Release

```
$ make release
```
